The InEvitables - NavUP

A Mobile Navigation and Orientation Application

  • Team Members
  • Team Resources

Similar to existing navigation applications such as Google Maps and Waze, the NavUP is a standalone mobile application that acts as a campus navigation system for students both with and without visual disabilities. The system consists of visual navigation using mobile networks, WiFi and GPS to provide accurate outdoor and indoor navigation, and a machine vision section which reads fiducial markers to help orient the visually disabled.
